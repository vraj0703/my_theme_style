# My Theme Style

A centralized package for app theming, typography, colors, and design tokens.

## Key Features

- **Design Tokens**: Standardized colors and fonts.
- **Theme Data**: Pre-configured `ThemeData` for light and dark modes.
- **Assets**: Shared JSON assets for style configurations.

## Getting Started

### Prerequisites

- Flutter SDK

### Installation

Add to your `pubspec.yaml`:

```yaml
dependencies:
  my_theme_style:
    git:
      url: https://github.com/vraj0703/my_theme_style.git
```

### Usage

Wrap your app with the provided theme builder or access constants directly.
